# three.js
## 动手写第一个Three.js程序

### 资料
- Three.js官网地址：http://threejs.org/
- Three.js库下载地址：https://github.com/mrdoob/three.js/
- Three.js核心库：three.js/three.min.js
- Three.js扩展库位置：three.js-rxx\examples\js
- 极客学院：jekexueyuan.com

### 概念
- 场景（Scene）：模型、灯光等
- 相机（Camera）：观察场景的视角
- 渲染器（Renderer）：场景渲染输出的目标
- 渲染（render）：执行渲染操作

### 添加
- 创建场景（scene）
- 添加网格模型（mesh）
- 添加灯光（light）
- 添加相机（camera）
- 创建渲染器（renderer）
- 渲染（render）
